## Running the application

### Setting up enviroment variables

In the [env.example](https://github.com/arthurhdr/TypescriptEcommerce/blob/main/.env.example) you'll have a template of what the .env file should look like. You must setup an account on Stripe, Prisma and Resend to use the app 

### Running

Open the terminal on the project folder and run

```bash
npm install
```

Then, wait for the modules installation. after that run

```bash
npm run dev
```

And go to the url http://localhost:3000 on your favorite browser.
